EE Product Listing app

Pages:

1. `/products` : Product list page
2. `/cart`: Cart Page

To run in dev mode:
`npm run dev`

To run in production mode
`npm run build && preview`

Run tests
`npm run test`

coverag: `npm run test:coverage`

Note:
Almost all the important files (components,Pages,context) are tested except for few items in(common/,layout)
